# Umami Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为首页 `index.html` 接入 Umami 访问统计脚本，并用现有测试方式验证脚本已正确写入。

**Architecture:** 保持纯静态站结构不变，先扩展一个针对首页 HTML 内容的轻量校验脚本，再在 `index.html` 的 `<head>` 中插入 Umami 官方脚本。实现只改动静态 HTML 与测试文件，不引入构建流程或运行时注入逻辑。

**Tech Stack:** HTML, Python 3, 纯静态文件部署

- Test: `tests/verify_homepage_card.py`

```html

  defer
  src="https://umami.aistar.cool/script.js"
  data-website-id="8f4c108a-d47c-4449-aa25-66abdbd8303e"
></script>
```

- [ ] **Step 2: 将脚本放在 `index.html` 的 `<head>` 内，靠近现有本地脚本标签**
