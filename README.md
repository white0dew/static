# 个人博客静态站（HTML/CSS/JS）

这是一个纯静态个人博客模板，适合部署到虚拟主机，结构简单、维护方便。

## 文件结构

```text
.
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── posts/
    ├── china-models-2026.html
    ├── global-models-engineering.html
    └── ai-model-roadmap.html
```

## 已包含内容

- 首页：个人介绍、文章列表、分类筛选（JS）
- 文章：3 篇 AI 主题文章（国内模型、国外模型、模型基础）
- 响应式样式：支持桌面与移动端
- 页脚备案位：可直接替换为真实备案号

## 部署方式

将整个目录文件上传到虚拟主机的网站根目录即可。
